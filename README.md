# arduino-autonomous-robot-cleaner
Arduino-based autonomous cleaning robot with motion control

## Project Overview
This project involves the design and implementation of an Arduino-based autonomous cleaning robot. The system integrates sensors, DC motors, and control logic to enable obstacle detection and autonomous navigation.

## Objectives
- Design a mobile robotic platform
- Implement obstacle detection using sensors
- Develop motion control logic for autonomous navigation
- Integrate hardware components with embedded programming

## Hardware Components
- Arduino microcontroller
- DC motors
- Motor driver module
- Distance sensor (ultrasonic sensor)
- Power supply system

## Software Implementation
- Developed control logic using C/C++
- Implemented obstacle detection and avoidance algorithm
- Controlled motor directions and speed via PWM signals
- Structured movement decision logic based on sensor input

## System Behavior
The robot detects obstacles in its path and adjusts direction accordingly. 
Basic autonomous navigation logic was implemented to allow continuous movement and area coverage.
